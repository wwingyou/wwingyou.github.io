*이제부터 너는 "MK-1"이다.*

일단은 서버로 사용할 기본 설정을 하기 위해 맥북에 연결해서 쓰던 트랙패드와 블루투스 키보드를 
연결해서 사용했지만 여간 불편한게 아니다(트랙패드를 리눅스에 연결해서 쓰는 것은 정말 최악의 
생각이다). 그래서 SSH 설정만 해놓고 나서는 거추장스러운 것들은 다 떼어버리고 켜놓기만 할것이다. 
그리고 SSH로 접속해서 사용할 생각이다. 

## SSH 서버 설치 

감사하게도 라즈베리파이 기본 OS인 Debian에 기본으로 SSH 서버 설정이 되어 있었다. 그래서 
따로 설치해줄 필요는 없었다. 

## 유저 생성

다음으로는 원격으로 접속할 유저를 생성해줬다. 루트 유저로 접속해도 되지만, 왠지 보안상 
좋지 않을것 같아 유저를 하나 만들기로 했다. 리눅스에서 유저 생성은 다음 커맨드로 할 수 있다.

```
$ sudo useradd mk1 -d /home/mk1
```

`mk1`이라는 유저를 만들고 홈 디렉토리를 `/home/mk1`으로 설정해줬다. 
그 다음은 패스워드 설정이다. 

```
$ sudo passwd mk1
```

그리고 홉 디렉토리의 소유주를 `mk1`으로 바꿔줬다.

```
$ sudo chown mk1 /home/mk1
$ sudo chgrp mk1 /home/mk1
```

여기서 끝이 아니다. `mk1`에게 sudo 권한을 부여해주어야 한다. sudo 권한 부여는 `/etc/sudoers` 
파일을 설정해주면 되는데, 중요한 것은 꼭 `visudo` 커맨드를 사용해서 수정해주어야 한다는 것이다.

```
$ sudo sudovi -f /etc/sudoers
```

그리고 아래 라인을 추가해줬다. 

```
mk1 ALL=(ALL) NOPASSWD: ALL
```

이렇게 하면 패스워드 입력 없이 sudo  커맨드를 사용할 수 있다.
